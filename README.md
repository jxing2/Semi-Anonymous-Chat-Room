This is a chat room application (both server and client) made for school classes.

Description:
In the class, some students are too shy to speak out their questions if they have
problems with the idea their teacher just taught them. 
So, this application allows them to ask questions anonymously, nobody's going to see their names at all.
However teacher have the right to check the chat log file which reveals students' real name, 
like teacher can see which student asked a certain question, this is why this application is called semi-anonymous chat root;
In addition, all teachers and students can share files through this application.
